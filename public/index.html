📖 Help Center & API Setup

Learn how WeatherMind uses API keys directly in your browser and how to get them for free.

Google Gemini 2.0 Flash Setup

Gemini is Google's flagship multimodal model. The free tier offers 1,500 requests per day.

  1. 1
    Visit Google AI Studio:

    Go to Google AI Studio 🔗 and sign in with any Google account.

  2. 2
    Create API Key:

    Click the blue "Get API key" button, then click "Create API key". You can choose to bind it to an existing Google Cloud project or auto-generate one.

  3. 3
    Copy the Key:

    Copy the generated key. It will start with AIzaSy....

  4. 4
    Save in App:

    Open the app's Settings, select the Gemini tab, paste the key, and click Save & Test.

⚠️ Region Restrictions: Google AI Studio free tier has geographic availability restrictions. If you reside in the EU, UK, or other restricted areas, the API call might fail. If so, we recommend using Groq instead!

Groq Open-Source Models Setup

Groq provides ultra-fast hosting for leading open-source models, including Llama (Meta) and Gemma (Google). Free tier offers 14,400 requests per day.

  1. 1
    Visit Groq Console:

    Go to Groq Console 🔗 and register for a free account (no credit card required).

  2. 2
    Create API Key:

    Navigate to API Keys in the sidebar, click "Create API Key", name it (e.g., "WeatherMind"), and click create.

  3. 3
    Copy the Key:

    Copy the key immediately as it will only be shown once. It will start with gsk_....

  4. 4
    Save in App:

    Open the app's Settings, select the Groq tab, choose a model (Llama 3.3 is recommended), paste the key, and click Save & Test.

How BYOK (Bring Your Own Key) Protects You

WeatherMind uses a serverless client-side architecture. This design values your privacy and saves hosting costs.

🔒

Direct Browser-to-AI Communication

Your API requests go directly from your web browser to Google or Groq servers. No middleman, database, or third-party server intercepts your messages.

💾

Secure Local Storage

Your API keys are stored solely inside your browser's local sandbox (using localStorage). They are never sent to WeatherMind or any external server other than the respective AI endpoints.

🆓

100% Free Hosting

Because the app doesn't require backend servers to pay for, we can keep this app open-source, fast, and completely free for anyone to use.

Frequently Asked Questions

Is it completely free?

Yes! Both Google AI Studio and Groq Console offer generous free tiers that cost $0. No credit card is required to sign up.

What if I get "Invalid API Key"?

Ensure you pasted the key into the correct tab: Gemini keys (starting with AIzaSy) go under Gemini; Groq keys (starting with gsk_) go under Groq.
